Scrapping for http://quotes.toscrape.com

Program works only in unix systems.
1. You need to install python version 3.5 or higher.
2. Run terminal in the program's directory
3. Type "python3 main.py"
4. Follow the instructions step by step:
    a). to scrapping webpage input 'y' and press 'Enter';
    b). after scrapping done, input 'n' and press 'Enter'
    c). if you need to finde info about authors, enter number of their + 'Enter'
    d). next step, enter ids (numbers 1-100) of interesting authors
    f). to exit input '0' + press 'Enter'
5. After program done, you can see all parsed data in '/parsed_data' directory
